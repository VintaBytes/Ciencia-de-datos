# Ciencia de Datos con Python
## (2025-2026)

Este repositorio contiene los cuadernos Colab más relevantes creados como material de apoyo para clases de programación y análisis de datos utilizando Python.

El material está dividido en secciones, con un índice en cada una de ellas.

---
## Resúmen del contenido:
### 1. Análisis de Datos con Python (2025)
Esta sección contiene cuadernos Colab útiles para aprender a realizar [Análisis de Datos con Python](https://github.com/VintaBytes/Ciencia-de-datos/tree/main/An%C3%A1lisis%20de%20Datos%20con%20Python#readme)

En ella encontrarás:
- Técnicas para limpiar datos con Pandas
- Selecciones y filtros de datos
- Funciones de Agregación y Agrupamiento de Datos
- Manejo de Arrays en NumPy
- Estadística descriptiva: Análisis de Tendencia Central
- Visualización de Datos (Matplotlib, Seaborn, Plotly)
- Análisis y Cálculo de Correlación
- Creación de Dasboards
- Etcétera.

---
### 2. Machine Learning con Python (2026)
Esta sección contiene una serie de cueadernos que te van a guiar para aprender [Machine Learning con Python](https://github.com/VintaBytes/Ciencia-de-datos/tree/main/Machine%20Learning%20con%20Python#README.md)

En ella encontrarás:
- Introducción al Machine Learning
- Repaso de Pandas, Matplotlib, Seaborn y Plotly
- Transformación de datos
- División de datos y validación cruzada
- Regresión lineal
- Regresión Logística y Evaluación de Clasificadores
- Clasificación con KNN (K-Nearest Neighbors)
- Arboles de Decisión
- Random Forest
- Aprendizaje no supervisado y clustering
- Automatización del flujo de trabajo con Pipelines y GridSearchCV
- Comparación y evaluación de modelos
- Pipelines
- Etcétera.
